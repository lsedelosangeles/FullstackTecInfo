## Concepto

El navegador web es una clase de aplicaciones que permite la interpretación y visualización de contenido web (**[hipertexto o hipermedia][g1]**) en un dispositivo. Aparte de interpretar la estructura del texto y mostrar las imágenes complementarias al mismo, los navegadores también renderizan el **[estilo][g2]** del documento, y ejecutan los ***[scripts][g3]*** que lo acompañan.

Inicialmente, los navegadores web solamente manejaban texto, ya que el propósito inicial de estos era la visualización de documentos académicos. Muy temprano en el desarrollo de la Web, se añadió la posibilidad de visualizar imágenes y establecer estilos para formatear el texto. Para mediados de los años 90 ya existían un número considerable de navegadores para acceder a la creciente oferta de contenido de la Web temprana.

Hoy en día, el funcionamiento de un navegador web esta determinado por un conjunto de estándares definidos por el ***[World Wide Web Consortium][l1]*** que establecen los parámetros que debería seguir esta clase de software para considerarse aceptable para su uso.

## Funcionamiento Básico

Un navegador web es, básicamente, un visualizador de documentos. Permiten visualizar el contenido, la estructura, y el estilo de documentos llamados **[páginas web][g4]**, las cuales se organizan en **[sitios web][g5]**. El navegador web facilita el acceso a estos, accediendo automáticamente al contenido de ambos, ya sea mediante referencias implícitas en el código de las páginas, o cuando el usuario activa un hipervínculo de las mismas.

Esta navegación queda registrada en un **historial**: un registro secuencial, por orden de visita, de las páginas que se han visitado.

En su forma básica, un navegador web moderno consiste de tres secciones:

![i1]

- **Viewport**: Donde el usuario visualiza e interactúa con el contenido de la página web actual. Ocupa la mayor parte de la ventana de la aplicación.
- **Barra de Direcciones**: Donde se ingresa y visualiza el **URL** de la página web actual.
- **Herramientas de Navegación**: Botones que permiten controlar la navegación entre páginas, de . Suelen ser los siguientes:

  - *Atrás*: Permite regresar a la última pagina visitada antes de la actual.
  - *Adelante*: Si se ha usado el botón *Atrás*, permite regresar a la página más reciente.
  - *Detener*: Permite detener la descarga desde el servidor, de la página que se esta accediendo (si no completó el proceso).
  - *Refrescar*: Permite volver a descargar la página actual desde el servidor. Muchos navegadores actuales combinan este botón con el anterior.
