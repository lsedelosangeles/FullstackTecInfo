## Página Web

Se denomina **Página Web** a todo documento redactado y diseñado para ser visualizado mediante un **navegador web**. Para ello, se debe redactar su contenido mediante **HTML**, y suele complementarse con **CSS** (para el manejo de la apariencia y la interactividad básica), y **JavaScript** (para agregarle funcionalidad y la interactividad avanzada).

[*Volver al Inicio*][inicio]

***

## Sitio Web

Se denomina **Sitio Web** a todo **conjunto de páginas web** y **recursos asociados** (imágenes, directorios, archivos, servicios, funcionalidades, etc.), que se encuentran alojados,y son accesibles, bajo un mismo [*nombre de dominio*](#dominio-o-nombre-de-dominio) o [*IP*](#ip-o-dirección-ip).

[*Volver al Inicio*][inicio]

## Diseño Web

Es el conjunto de conceptos, estándares, protocolos, metodologías, y herramientas que permiten el desarrollo de páginas y sitios web.

## Programación Web

Es el conjunto de conceptos, estándares, protocolos, metodologías, y herramientas que permiten el desarrollo de **aplicaciones y servicios**, de forma que los mismos sean **accesibles mediante un sitio web**.

## Glosario

### Dominio, o Nombre de Dominio

Se refiere a la dirección única que identifica a un sitio web en Internet. Esta compuesto por un **nombre de segundo nivel** (ejemplo: `www.google`, `www.ceibal`, etc.), *que se indica primero*, y un **dominio de nivel superior** (ejemplo: `.com`, `.com.uy`, `.edu.uy`, `.org`, etc), *que se indica por último*. Los dominios deben adquirirse (más allá de que algunas plataformas ofrezcan sub-dominios gratuitos bajo ciertas condiciones), son otorgados por **organismos reguladores**, ante los cuales deben solicitarse (como ser el [*Servicio Central de Informática de la UdelaR*][l1] en Uruguay), y son manejados a nivel global por el [**DNS**](#dns)

### IP, o Dirección IP

Se refiere al *identificador único* que recibe un *dispositivo* al conectarse a una red, ya sea de forma local (en una [**LAN**](#lan)), o en Internet. Si el dispositivo aloja algún servicio y lo comparte en esa red, su IP se transforma, además, en un nombre de dominio bajo el cual se puede acceder a dicho servicio.

### LAN

Significa *Local Area Network* (en español: *Red de Área Local*). Se refiere a cualquier red de computadoras que abarque un área no mayor a la de un hogar, una oficina o un edificio. Una LAN **no está, necesariamente, conectada a Internet**. De ser así, lo hace mediante un [**router**].

### DNS

Significa *Domain Name System* (en español: *Sistema de Nombres de Dominio*). Se refiere a un sistema de servidores y bases de datos que asocia un **nombre de dominio** a una **dirección IP** específica.

### Router o Enrutador

Se trata de un dispositivo de red que permite a los dispositivos de una red (por ejemplo, una LAN), comunicarse con los dispositivos de otra red (por ejemplo, Internet).
